Full page data selector. This component will populate the sidebar with a select card component to let the user select a dataset. The dataset is presented on a full page including preview and data tables. Once a dataset was selected, the component will collapse into a dropdown in the sidebar (or any other container). The component is controlled via session_state:

  • story_mode can be set to False to always force component collapse
